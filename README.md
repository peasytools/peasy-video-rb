# peasy-video

[![Gem Version](https://badge.fury.io/rb/peasy-video.svg)](https://rubygems.org/gems/peasy-video)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Video processing library for Ruby — trim, resize, rotate, extract audio, generate thumbnails, convert to GIF. FFmpeg-powered with a clean Ruby API.

Built from [Peasyvideo](https://peasyvideo.com), a free online toolkit.

## Install

```bash
gem install peasy-video
```

## Quick Start

```ruby
require "peasy_video"

# Check version
puts PeasyVideo::VERSION  # => "0.1.1"
```
